Instruction to extract rosbag in csv format and plot them using Matlab.

1. 	Record rosbag containing topic "piksi/debug/receiver_state"
2.	Move in the folder containing the folder and execute the "bag_to_csv.py" script 
	(taken from https://www.clearpathrobotics.com/assets/guides/ros/Converting%20ROS%20bag%20to%20CSV.html)
	and execute "python bag_to_csv"
3.	Run "example_print_cn0.m" in Matlab.

For furtherquestions or issues please use github "Issue" section :)
